Setup

React frontend is not uploaded

This server serves your SPA from a production build. Upload your React build output so index.html is available and the app loads properly.

  1. 1
    Build the React app In your frontend project, run npm run build (or yarn build / pnpm build).
  2. 2
    Copy artifacts into build/ Place the build output next to the API so this file exists: build/index.html
  3. 3
    Reload Restart the API if needed, then open this URL again — the React app will be served from the root path.
